# Conversion Function

## Problem

```text
Create a function that, given a string of conversions, and a value, and a target unit, gives the amount
in that target unit.

1 kg = 1000 g
1 mile = 1.609 km
1 km = 1000 m
1 m = 100 cm
1 cm = 10 mm
1 pound = 0.454 kg

def fn(conversions: str, unit: str, amount: float, target_unit: str)
-> float
```

## Run Solution

```shell
python3 function.py
```
